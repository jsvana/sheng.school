# sheng.school

A deliberately cheesy, school-project-style website for the **Quansheng K1 CW Trainer** —
firmware that turns a ~$20 Quansheng UV-K1 / UV-K5 V3 handheld into a dedicated
Morse code (CW) learning machine.

**Live:** https://sheng.school

Static single-page site (`index.html`), served via GitHub Pages.

The trainer firmware lives in its own repo:
[uv-k1-k5v3-firmware-custom](https://github.com/jsvana/uv-k1-k5v3-firmware-custom).
